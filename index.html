Foucault Pendulum

by Nate Beatty

Simulation of a foucault pendulum as if it started swinging at an angle of 15 degrees in the north-south plane relative to the earth's position at the start of the day today.

Enter your zip code in the box below to see the real-time simulation of the pendulum as if it were positioned at your current location. The default location, loaded when the page loads, is at the north pole.

This site was built using three.js and the Google Geolocate API. Background image by Jochum Skoglund. Simulation requires WebGL and a modern browser to run properly. Thanks for visiting!
